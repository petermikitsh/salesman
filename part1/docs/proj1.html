CSCI-261 Programming Project

Optimal Traveling Salesman Problem

Part 1 Due: Sunday September 15th, 2013 by 11:59:00pm

Program Requirements

Generate an optimal tour of the TSP by considering all possible permutations of cities. A tour will always begin and end with vertex 0. Compute the cost of each permutation to find the tour of least cost. If n <= 10 print out the list of X-Y coordinates, and the graph as an adjacency matrix of Euclidean weights. If n <= 5 print out the list of all possible paths, in lexicographic order (a path will always begin and end with a '0'). For all values of n print out the optimal distance, the optimal path, and the runtime for the algorithm.

Error Conditions and Messages

  1. Missing one or both command line arguments:
    • Usage: java OptimalTSP n seed
  2. One or both command line arguments is not an integer:
    • Command line args must be integers
  3. Value of n is less than 1 or greater than 13:
    • Number of vertices must be between 1 and 13

Your program should print the error message to standard output (not standard error) and terminate upon any of these error conditions. No other error condition will be tested for.
